Permite analizar entre varias opciones posibles. El carácter de alternancia es el |. Ejemplos de sintaxis para alternancia entre opciones: [uno|dos|tres] [a|e] [hombre|mujer]

Validar el ingreso de un típo y nro de factura. Para que sea correcto el primer carácter debe ser a,b o c y el nro debe tener uno o más dígitos.
